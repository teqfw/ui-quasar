## Overview

Plugin for [Quasar UI](https://quasar.dev/) components to use them in Tequila Framework based apps.

### Install

```shell
$ npm i @teqfw/ui-quasar --save 
```

### Namespace

This plugin uses `TeqFw_Ui_Quasar` namespace.
